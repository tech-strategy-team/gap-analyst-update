# Github issue

![issue](https://github.com/images/modules/site/issues/fp24/features-river-1.webp)

## 概要

リポジトリに問題を作成して、作業の計画、議論、追跡を行うことができます。 issue は迅速に作成でき、柔軟性があり、さまざまな方法で使用できます。 issue は、バグ レポート、新機能やアイデア、およびチームに書き留めたり、話し合ったりするために必要なものなどを追跡できます。 sub-issue を追加することで作業をさらに細分化し、実行する作業の完全な階層を簡単に閲覧することもできます。

## 作成方法

- [GitHub Issueのクイックスタート](https://docs.github.com/ja/issues/tracking-your-work-with-issues/configuring-issues/quickstart)
