# PowerPoint to Markdown Converter

このプロジェクトは、PowerPointファイル（.pptx）をMarkdown形式に変換するためのPythonスクリプトを提供します。スライド内のテキストを抽出し、Markdown形式で保存します。

## 機能
- PowerPointファイルのスライドごとにテキストを抽出。
- 抽出したテキストをMarkdown形式で保存。

## 必要条件
以下のソフトウェアとライブラリが必要です。

1. **Python**（バージョン3.6以上推奨）
2. Pythonライブラリ：
   - `python-pptx`

## インストール手順

### 1. Pythonのインストール
- Pythonがインストールされていない場合は、[公式サイト](https://www.python.org/)からダウンロードしてインストールしてください。
- インストール後、コマンドラインで以下のコマンドを実行して、インストールが成功していることを確認します：
